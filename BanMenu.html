About
During the COVID pandemic there was an explosion in SSH connection attempts. A big list didn't give much insight. So I created a few (python) scripts for generating a JSON file which I could feed to a circlepacking module.
This is the result.
Circles are zoomable, at the non-world levels dragable.
A mouseover reveals textual info.
